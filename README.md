## 📝 Practice Problems
### 1️⃣ Find Maximum and Minimum in an Array
정수 여러 개를 입력받아 배열에 저장한 뒤,  
배열의 **최대값과 최소값**을 각각 함수로 구하는 프로그램을 작성한다.

**학습 포인트**
- 배열 사용
- 함수 분리
- 지역 변수 개념
- 반복문 활용
------------------------------------------------------------------------------------------------------------------------------------------
### 2️⃣ Student Score Average and Grade
학생 수 `n`을 입력받아 `n`명의 점수를 배열에 저장한다.  
모든 점수의 **평균을 계산**하고, 평균에 따라 **학점(A~F)** 을 출력한다.

**학점 기준**
| Average | Grade |
|------|------|
| 90 이상 | A |
| 80 이상 | B |
| 70 이상 | C |
| 60 이상 | D |
| 60 미만 | F |

**학습 포인트**
- 배열을 함수로 전달
- 함수의 반환값
- 조건문을 이용한 등급 판별

------------------------------------------------------------------------------------------------------------------------------------------

### 3️⃣ Reverse Array Elements
정수 `n`개를 입력받아 배열에 저장한 후,  
배열의 원소를 **역순으로 뒤집는 프로그램**을 작성한다.

**조건**
- 추가 배열 사용 금지
- 배열 뒤집기 기능은 함수로 구현

**학습 포인트**
- 배열 인덱스 이해
- 반복문과 임시 변수 활용
- 함수에서 배열 수정
